Perfect Home

Perfect Home is a website for rent and lease houses through online. It developed by using HTML, CSS, JavaScript for front end and PHP, MySQL for back end. The website consist of 3 modules user, house owner and admin.

User

  • User can sign up and login to the system
  • User can rent and lease houses
  • User can pay the rent through the website
  • User can give feedbacks about the house and the owner

Owner

  • Owner can sign up and login to the system
  • Owner can add houses for rent and lease
  • Owner can approve the request from the user for rent and lease
  • Owner can view the feedbacks of the user

Admin

  • Admin can view the list of users and owners and approve them
  • Admin can view the list of houses and approve it
  • Admin can view the feedbacks and complints of the users

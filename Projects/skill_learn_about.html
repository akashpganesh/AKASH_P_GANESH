Skill Learn

Skill learn is a mobile application for learn IT Skill. It is developed by using Flutter as front end and NodeJS, MongoDB as back end. It has consist of 3 modules User, Tutor, Admin

User

  • User can sign up and login to the application
  • User can enroll into a course and learn through videos
  • User can access the videos and document in the course after enrollment
  • User can chat with the tutor and other students through the chat group
  • User can ask doubts through the comment section of the video

Tutor

  • Tutor can sign up and login to the application
  • Tutor can add courses to the application
  • Tutor can add videos and documents to the course
  • Tutor can chat with the students through the chat group

Admin

  • Admin can view the list of users ang tutors and approve them
  • Admin can view the list of courses and approve it
